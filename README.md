# TF2 CUSTOM FOLDER UTILITY

TF2 custom folder Utilty (in short TCFU) was created for makeing some of the task easier with just 1 click

---
# TFCU FEATURES
- Cache files Remover
- Mod Limit Fixer (by [HammerHead133](https://gamebanana.com/tools/16705))
- Easy to read config file to change things like:
- Auto start setting (automatically starts with centran mode)
- Language setting for mod limit fixer

---
## Instaltion
See [Releases](https://github.com/OHsit/TF2-Custom-Folder-Uitlity/releases) and download the latest version for Windows
Unpack the zip
Put the folder in the custom folder and run .bat file
